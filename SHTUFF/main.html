The purpose of this website is to compare gene annotations between Tarsius_syrichta.tarSyr1.75.gtf from ENSEMBL to Tarsius_syrichta.tarSyr1.86.gtf from ENSEMBL for the species Tarsius syrichta.

The homepage, which you are currently on, provides access to the overview and comparison pages.

The Overview page lists summary information for genes and transcripts in each release in comparison.

The Comparison page can query our database and compare the genes and transcripts for each release.
